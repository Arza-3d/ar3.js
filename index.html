construct table

construct table from JSON (1st)

ar3-JSON-table-1.js

Create table from JSON file. Have more control over what members of the JSON object, and have more control over the table. Use it for JSON and table that you wanna keep edit in the future.

JSON file must use variable name: _JSON_string_r3
I add _r3 suffix to evade possible naming collision, since this variable will be exposed as global variable.
Example:

var _JSON_string_r3 =
`
{
  "table1":
  [
    {
      "col1":"row1c1",
      "col2":"row1c2"
    },
    {
      "col2":"row2c2",
      "col1":"row1c2"
    }
  ],
  "table2":
  [
    {
      "col1":"row1c1",
      "col1":"row1c2"
    }
  ]
}
`;

example of the HTML table structure:

<table data-JSON-r3="table1>
  <thead>
    <tr>
      <th rowspan="2">line 1</th>
    </tr>
    <tr>
      <th data-JSON-r3="col1">th name 1</th>
      <th data-JSON-r3="col2">th name 2</th>
    </tr>
  </thead>
</table>

C++ code wrapper

UE4 C++ code wrapper

ar3-cpp-ue4-code-wrapper-1.js

Automation for text wrapping C++ code example for UE4 documentation. This are things that the code will do:

  1. define the tag target with class cpp-codex-r3,
    then do this in target text:
    1. change < to &lt;, then > to &gt;
    2. wrap the base class defined in attribute data-cpp-class-r3
    3. wrap the highligted function defined in attribute data-cpp-funct-r3
    4. wrap the custom variable defined in attribute data-cpp-var-r3
    5. wrap the optional custom variable defined in attribute data-cpp-var2-r3
    6. wrap the input string defined in attribute data-cpp-input-r3
  2. then do this for other trivial things:
    1. automatically wrap comment
    2. automatically add tab
    3. change newline with <br>

    usage example:

    <pre class="cpp-codex-r3"
      data-cpp-class-r3="BaseClass"
      data-cpp-funct-r3="Function_1,Function_2,Function_n"
      data-cpp-var-r3="Variable_1,Variable_2,Variable_n"
      data-cpp-var2-r3="Variable_a,Variable_b,Variable_c"
      data-cpp-input-r3="Input_1,Input_2,Input_n"
    >
      // code example 1
    </pre>
