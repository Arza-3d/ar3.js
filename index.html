ar3.js

javaScript for my personal documentation

The reason why I am making this js & css files, is to help me take notes for my game development study. I am learning Unreal Engine 4, and they are using visual scripting, which means I need to take lot of huge screenshot of the visual script during my study. I was gonna make study notes in pdf, but I didn't think it will work well, and if I just scatter it in jpg it will be a mess...

So my solution is to take study notes in html and make my own js & css that fit my criteria of good study notes.
My preferences for good study notes are...

  • Small readable font
  • Active navigation
  • Hidable element
  • Plus you can embed video in html notes

Since my script that was meant to help me take study notes, getting bigger and bigger, and it's kinda getting messy, I need to make the documentation for the script. So this page will be documentation of ar3.js and ar3.css.

Automate trivial things

Construction phase only will be done once, at the start of web load, and that's it. If people download the page and open the offline version than the construction will not be executed.

add necessary attributes:

  1. Make sure the page is scaled for mobile device
  2. Make sure <i> has attribute lang="en", if the page is not in English. The other language that sometimes I use is in the documentation is Indonesian language, ex: <body lang="en">
  3. Since it's just code, don't translate <code> and <pre>...
    $('code, pre').attr('translate', 'no');
  4. ...but translate it if it has a comment (has comment-r3 class):
    $('.comment-r3').attr('translate', 'yes');
  5. Make sure every videos have control:
    $('video').attr('controls');
  6. Make sure outside links open new tab:
    $("a[href^='http']").attr('target', '_blank');

add necessary elements:

  1. Make sure <table> is inside element with style overflow:auto...
    $('table').wrap('<div style="overflow:auto">');
