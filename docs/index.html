The reason why I am making this js & css files, is to help me take notes for my game development study. I am learning Unreal Engine 4, and they are using visual scripting, which means I need to take lot of huge screenshot of the visual script during my study. I was gonna make study notes in pdf, but I didn't think it will work well, and if I just scatter it in jpg it will be a mess...

So my solution is to take study notes in html and make my own js & css that fit my criteria of good study notes.
My preferences for good study notes are...

  • Contain lot information in a page, just like books
  • Small readable font, instead of big fonts
  • Hidable element, cause I don't like long scroll
  • Plus you can embed video in html notes

Since my script that was meant to help me take study notes, getting bigger and bigger, and it's kinda getting messy, I need to make the documentation for the script. So this page will be documentation of ar3.js and ar3.css.

short pre

shorting code

I usually use this to show two version of same code, the short one just showing the important part of the code, and the long one showing the fully working code.

Every two consecutive <pre> tags, something like this; <pre>short code</pre> <pre>long code</pre> will be automatically made into short code, like this example:

So here is the code:

jQuery + CSS

Here is the link javaScript ES6, or this is the javaScript ES5. 

{ // ar3-short-code 
    const constructNote = 'r3-short-code_done', 
        isConstructed = $('body').hasClass(constructNote); 
 
    { // the main construction of short code 
        let text1 = 'show all code', 
            text2 = 'short code'; 
 
        if (!isConstructed) { 
            $('pre + pre').css('display', 'none') 
                .before('<button class="r3-short-code r3-active">'+ text1 + '</button>'); 
        } 
 
        $('.r3-short-code').click(function() { 
            if ($(this).hasClass('r3-active')) { 
                this.innerHTML= text2; 
                $(this).prev().delay(300).toggle(600); 
                $(this).next().toggle(300); 
            } else { 
                this.innerHTML= text1; 
                $(this).prev().toggle(300); 
                $(this).next().delay(300).toggle(600); 
            } 
            $(this).toggleClass('r3-active'); 
        }); 
    } 
 
    $('body').addClass(constructNote); 
}

By default the page is English, if it is Indonesia than make sure you have this <body lang="id">, here is the example:

Here is the code and the sources javaScript ES6 / javaScript ES5. 

const isIndo = $('body').attr('lang') == 'id';
let text1 = isIndo ? 'tunjukkan semua kode' : 'show all code',

        isConstructed = $('body').hasClass(constructNote);

            text2 = isIndo ? 'persingkat kode' : 'short code';

short code (3rd update)

So this is a scenario if I wanna change the button text globally, for example, if somehow I wanna take notes not in English or Indonesia.

To do this, I put the new text in <body> attribute, like this <body data-short-code-r3-text1 = "new text 1" data-short-code-r3-text2 = "new text 2">
Here is an example:

Here is the code and the sources javaScript ES6 / javaScript ES5.

let newText1 = $('body').attr('data-short-code-r3-text1'), 
    newText2 = $('body').attr('data-short-code-r3-text2'); 
if (newText1 !== undefined) {text1 = newText1;} 
if (newText2 !== undefined) {text2 = newText2;}

short code (4th update)

The last override is in case I wanna change the button text unique to the individual <pre> that is targeted.

To do this I put the new text as an attribute of the 2nd <pre>:<pre>short code</pre>
<pre data-r3-text1 ="new text 1a" data-r3-text2 = "new text 2a">long code</pre>

Here is an example:

Here is the sources javaScript ES6 / javaScript ES5. and the ES6 preview:

// assigning text in button will be done with loop instead auto assign with jQuery 
const $pre = $('pre + pre'); 
if (!isConstructed) {$pre.css('display', 'none');} 
 
for (let i = 0; i < $pre.length; i++) { 
    { 
        let newerText1 = $($pre[i]).attr('data-r3-text1'), 
            newerText2 = $($pre[i]).attr('data-r3-text2'), 
            newestText1 = (newerText1 === undefined) ? newText1 : newerText1, 
            newestText2 = (newerText2 === undefined) ? newText2 : newerText2; 
 
        if (!isConstructed) { 
            $($pre[i]).before('<button class="r3-short-code r3-active">'+ 
                newestText1 + '</button>');

                this.innerHTML= newestText1;
