The reason why I create this js & css files, is to help me take notes for my game development study. I was learning Unreal Engine 4, and they are using visual scripting, which means I need to take lot of huge screenshot of the visual script during my study. I was gonna make study notes in pdf, but I don't think it will work well, and if I just scatter it in jpg it will be a mess...

So my solution is to take study notes in html and taylored my own js & css that fit my criteria of good study notes.
My preference for good study notes is...

  • Contain lot information in a page, instead of splitting it in pages
  • I hate big fonts
